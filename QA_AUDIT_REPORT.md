## CRITICAL BUGS (Fake data shown to users as REAL)

### C1. WhatsApp Service is MOCK — Messages NEVER Actually Sent
**File:** `backend/app/services/whatsapp.py:11-20`  
**Impact:** ALL WhatsApp features broken in production  
```python
@staticmethod
async def send_template(phone: str, template: str, variables: dict) -> str:
    """Mock WhatsApp Cloud API call."""
    print(f"[WhatsApp] Sending {template} to {phone}: {variables}")
    return f"wamid.{uuid.uuid4()}"  # FAKE message ID
```
**Affected features:**
- WhatsApp Send (`/whatsapp/page.tsx`) — appears to send but doesn't
- Follow-up Dispatch (`/followups.py dispatch-due`) — schedules but never delivers
- CRM Inbox replies (`/crm/inbox/page.tsx`) — replies go nowhere
- Auto follow-up service — deducts credits but messages never sent

**Fix needed:** Connect to real WAHA API or WhatsApp Cloud API (graph.facebook.com)
